About

A project by Austin Delamar based off of Dave Gamache's work and other contributors.

This really started as an extension of Skeleton for my porfolio site. But as I've added new features to my personal extension, I've wanted to make an easy way to centralize and share it.

If you'd like to contribute, feel free to fork and make changes, then open a pull request to master branch.
